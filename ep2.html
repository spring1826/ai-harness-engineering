2편 규칙

고삐 하나면 달라져요

마케팅팀이 AI에게 광고 문구를 요청했어요.

어제는 3개를 알려줬는데, 오늘은 5개.
형식도 매번 달라서 그대로 쓸 수가 없었어요.

월요일: 번호 없이 줄글로

화요일: 번호 매기고 설명까지

수요일: 갑자기 영어 섞어서

AI는 같은 질문에도 매번 다른 답을 해요.
고삐 없는 말과 같아요.

마구간 벽에 설정표를 걸어두면
누가 말을 몰아도 같은 방향으로 가요.

🪢 고삐 설정표
  • 방향은 항상 동쪽
  • 속도는 보통 걸음
  • 갈림길에서는 오른쪽
  • 쉬는 곳은 2시간마다

AI에게도 이런 설정표를 만들어주면,
매번 같은 품질의 답이 나와요.

실제 용어

Claude에서는 이 설정표를 CLAUDE.md라고 불러요.
.claude/ 폴더 안에 넣어두면 돼요.

규칙서를 만들면 이렇게 돼요

📋 AI의 규칙서
  • 항상 한국어로 답변
  • 전문 용어 대신 쉬운 말
  • 방법은 항상 3가지로
  • 결과는 표로 정리
AI 답변
방법 설명 난이도
직접 작성 팀에서 바로 쓸 수 있는 문구 쉬움
템플릿 활용 기존 양식에 맞춰 생성 보통
A/B 비교 두 가지 버전을 함께 제시 보통

같은 규칙서가 있으니, 누가 질문해도 같은 형식으로 답해요.

실전: AI에게 이렇게 말하세요

프롬프트
우리 팀에 맞는 CLAUDE.md 규칙서를 만들어주세요. 우리 팀은 [예: 마케팅팀]이고, 주로 하는 업무는 [예: 광고 문구 작성, 캠페인 기획]입니다. AI가 답변할 때 지켜야 할 규칙: - 사용 언어: 한국어 - 대상 독자: [예: 비개발자 / 경영진 / 고객] - 톤: [예: 친근하게 / 격식있게 / 간결하게] - 반드시 포함할 것: [예: 항상 근거를 제시 / 표로 정리 / 3가지 옵션 제시]
예시 프롬프트는 정답이 아닙니다. 참고만 해주세요. 상세하게 꼼꼼하게 전달할수록 더 좋은 결과물이 생성됩니다.

나중에 한 줄씩 추가하면 돼요. [대괄호] 안만 바꾸면 돼요.

한 번 적어두면 알아서 지켜요.
