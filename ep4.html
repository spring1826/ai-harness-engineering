한마디면 알아서 달려요

요청을 접수할 때마다
빠뜨리는 항목이 달랐어요.

월요일: 요청팀은 적었는데 마감일을 빼먹음

화요일: 마감일은 적었는데 담당자를 빼먹음

수요일: 전부 적었는데 우선순위를 안 정함

마차가 갈림길에 설 때마다
"왼쪽! 아니 오른쪽! 잠깐, 직진!"
매번 하나하나 지시하는 것과 같아요.

경로를 정해두면 한마디로 끝

😰
매번 하나하나 지시
"요청팀 이름 적어줘"
"마감일도 넣어줘"
"담당자 정해줘"
"우선순위 매겨줘"
"요약도 해줘"
🗺️
"A코스!" 한마디
경로가 정해져 있으니
한마디면 알아서
순서대로 달려요.
빠뜨리는 것도 없어요.

자주 하는 작업을 경로(세트 명령)로 만들어두면,
매번 설명할 필요가 없어요.

실제 용어

Claude에서는 이런 경로를 Skills(스킬)이라고 불러요.
/명령어(Slash Command)를 입력하면 바로 실행돼요.

/접수 한마디로 시작

사용자 입력
/접수
AI 자동 실행

요청 접수를 시작할게요.


1. 요청팀은 어디인가요?

2. 어떤 작업을 자동화하고 싶으신가요?

3. 마감일이 있나요?

4. 현재 이 작업에 얼마나 시간이 걸리나요?


답변해주시면 요청서를 자동으로 정리해드릴게요.

입력 한 줄로 AI가 정해진 순서대로 질문해요.
빠뜨리는 항목이 없어요.

실전: 2단계로 경로 만들기

1
Plan 모드에서 방법 물어보기 (Shift + Tab)
지난 작업과 문서를 확인해주세요. 아래와 같은 반복 작업을 한 번에 묶어서 자동으로 진행하고 싶습니다: 1. 요청자에게 필요한 정보를 순서대로 질문 2. 답변을 정해진 양식으로 정리 3. 정리한 내용을 문서로 저장 이렇게 반복작업을 묶음으로 자동화하려면 어떤 방법으로 적용할 수 있는지 방법을 고민하고 알려주세요.
예시 프롬프트는 정답이 아닙니다. 참고만 해주세요. 상세하게 꼼꼼하게 전달할수록 더 좋은 결과물이 생성됩니다.
2
방법을 확인하고 실행 맡기기
확인했습니다. AI가 기획한 대로 진행해주세요.

/접수만 입력하면 AI가 정해진 경로대로
순서대로 질문하고, 정리하고, 저장해요. 매번 같은 품질로.

이런 경로들을 만들 수 있어요

/접수
→
요청 정리
/분석
→
기획서
/비교
→
옵션 비교
/확정
→
실행 계획

경로를 만들어두면 한마디로 복잡한 작업이 시작돼요.
